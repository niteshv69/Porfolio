# Software Engineer Portfolio

Read-only animated portfolio website built with Vite (vanilla HTML/CSS/JS).

## Features

- Premium single-page portfolio layout
- Smooth scroll and section reveal animations
- Layered gradient + grid background effects
- Fully responsive desktop/mobile design
- Read-only contact section (no form submit/backend)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Customize

- Update name/title/content in `index.html`
- Update colors and motion in `style.css`
- Update section reveal behavior in `main.js`
